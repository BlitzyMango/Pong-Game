# Classic Pong Game
### Overview
A simple game of Pong developed using the pygame module on Python.

## Prerequisites
#### Minimum Python version(s)
-Python 3.3 or Python 3.4 (preferred)  
#### Pip installations:
- [Pygame](https://pypi.org/project/pygame/)
---
pip install pygame
---
- [Asyncio](https://pypi.org/project/asyncio/)
---
pip install asyncio
---
